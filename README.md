# SmartList

SmartList es una aplicación móvil Android desarrollada en Kotlin que permite a los usuarios gestionar listas de compras de forma simple e intuitiva. La aplicación cuenta con autenticación de usuarios mediante Firebase y permite crear, editar y completar listas de compras.

---

## Funcionalidades Principales

- Registro e inicio de sesión de usuarios
- Autenticación con Firebase Authentication
- Creación, edición y eliminación de listas de compras
- Gestión de productos dentro de cada lista
- Marcado de productos como comprados
- Separación de listas pendientes y completadas
- Interfaz simple y clara con navegación intuitiva

---

## Tecnologías Utilizadas

- Android Studio
- Kotlin
- Firebase Authentication
- Firebase Firestore / Realtime Database
- Gradle (Kotlin DSL)

- Git y GitHub

---

## Requisitos Previos

Antes de compilar el proyecto, asegúrate de tener instalado:

- Android Studio (versión reciente)
- JDK 11 o superior
- Conexión a internet
- Una cuenta de Firebase

---

## Clonar y Ejecutar el Proyecto

### Clonar el repositorio (desde cmd elegir una dirección)

```bash
git clone https://github.com/S3bas77/SmartList.git
cd SmartList
```

---

### Abrir el proyecto en Android Studio

- Abrir Android Studio
- Seleccionar Open
- Elegir la carpeta SmartList
- Esperar a que Gradle sincronice el proyecto

---

## Configuración de Firebase (IMPORTANTE)

El archivo de configuración de Firebase no está incluido en el repositorio por motivos de seguridad.

### Pasos:

1. Descargar el archivo google-services.json
2. Copiar o arrastrar el archivo en:

```
SmartList/app/google-services.json
```

3. Sincronizar Gradle nuevamente

---

## Ejecutar la Aplicación

- Conectar un dispositivo o emulador
- Presionar Run en Android Studio
